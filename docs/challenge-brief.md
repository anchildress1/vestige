# Gemma 4 Challenge — Cliff Notes

**Source:** https://dev.to/devteam/join-the-gemma-4-challenge-3000-prize-pool-for-ten-winners-23in

## Dates
- Opens: May 6, 2026
- Deadline: May 24, 2026, 11:59 PM PDT
- Winners announced: June 4, 2026

## Prizes — $3,000 / 10 winners
- **Build with Gemma 4** (5 winners): $500 + DEV++ + badge each
- **Write About Gemma 4** (5 winners): $100 + DEV++ + badge each
- All valid submissions get a completion badge
- Each submission picks ONE category, not both

## Build prompt
"Build something useful or creative with any Gemma 4 model. The scope is wide open — IoT integration, multimodal tool, long-context reasoning app. What matters is that Gemma 4 is doing real work at the heart of your project." Compelling entries show *why* the model was chosen and what it unlocked.

## Write prompt
"Publish a post about Gemma 4 that educates, inspires, or sparks curiosity." How-tos, comparisons, opinion essays, or deep technical breakdowns. If primarily showing a project, submit to Build instead.

## Gemma 4 model variants
- 2B / 4B effective params — mobile, edge, browser
- 31B dense — heavy lifting
- 26B A4B MoE — high-throughput, advanced reasoning
- Submission template uses the labels **E2B, E4B, or 31B Dense**

## Judging — Build
1. Intentional + effective use of the chosen model
2. Technical implementation / code quality
3. Creativity + originality
4. Usability / UX

## Judging — Write
1. Clarity + depth
2. Originality of perspective
3. Practical value
4. Quality of writing

`devchallenge`, `gemmachallenge`, `gemma`
