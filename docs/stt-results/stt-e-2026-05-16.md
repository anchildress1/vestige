## Raw evidence

### Q_aftermath

```
baseline top-5: [A3, X2, D3, B3, A5]  relevant=2/6
hybrid   top-5: [X2, A3, A2, D3, A1]  relevant=3/6  novel=[A2, A1]
```

Hybrid surfaces A2 ("Tuesday meeting. Same thing.") and A1 ("Standup ran long again.") which
the tag-only path missed. Both are aftermath/meeting-drain entries without the exact tag overlap
the query keywords match.

### Q_invoice

```
baseline top-5: [B3, B2, B1, A5, D3]  relevant=3/3
hybrid   top-5: [B2, B1, B3, A1, A5]  relevant=3/3  novel=[]
```

Tag+keyword already surfaces all three invoice entries. Hybrid reranks within the set but adds
no novel relevant entries → tie by definition.

### Q_decision

```
baseline top-5: [C1, A5, D3, B3, D2]  relevant=1/3
hybrid   top-5: [C1, C2, A1, D2, A5]  relevant=2/3  novel=[C2]
```

Hybrid promotes C2 ("Task app decision still looping.") — semantically relevant via embedding
similarity, not a strong keyword/tag match for "stuck choosing between options."

### Q_lateNight

```
baseline top-5: [D1, B2, B1, A1, A5]  relevant=1/3
hybrid   top-5: [D1, B2, A1, B1, D2]  relevant=2/3  novel=[D2]
```

Hybrid promotes D2 ("2:48am and I'm deciding whether to rebuild the folder structure.") —
thematically matched to the 3am rearranging query via embedding; missed by keyword scoring alone.
